HTML

HTML stands for Hypertext Markup Language. It's a standard markup language for creating web pages. HTML is a text-based approach for describing how content is structured within an HTML file. It tells a web browser how to display text, images, and other forms of multimedia on a webpage

CSS

CSS stands for Cascading Style Sheets. It's a computer language used to structure and lay out web pages. CSS is a stylesheet language that helps designers customize the appearance of websites

javascript

JavaScript is a high-level, interpreted programming language. It is one of the three core technologies of the World Wide Web, along with HTML and CSS. JavaScript is used to make web pages interactive and dynamic, and to provide functionality for a wide range of web applications.
